# Object-Oriented Programming (OOP)

## What is OOP?
OOP is a programming paradigm based on objects.

---

## Encapsulation
- Binding data and methods together
- Achieved using private variables and public methods

### Benefits:
- Data security
- Better control

---

## Inheritance
- One class acquires properties of another
- Achieved using `extends`

---

## Polymorphism
- One method, multiple behaviors
- Method overloading
- Method overriding

---

## Abstraction
- Hiding implementation details
- Achieved using abstract classes and interfaces
